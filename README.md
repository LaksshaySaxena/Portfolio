# Lakshay Saxena Portfolio

A modern personal portfolio website built with Next.js, Tailwind CSS, Framer Motion, and Three.js.

## Features

* Responsive Design
* Interactive Hero Section
* Skills Showcase
* Project Gallery
* Contact Form with EmailJS
* 3D Earth Animation
* Resume Download

## Tech Stack

* Next.js
* React
* Tailwind CSS
* Framer Motion
* Three.js
* EmailJS
